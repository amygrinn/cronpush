# Cron Push

## Project setup
1. Install node dependencies
```
yarn install
```

2. Start mysql docker container
```
docker-compose up -d
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Run your unit tests
```
yarn test:unit
```

### Run your end-to-end tests
```
yarn test:e2e
```

### Lints and fixes files
```
yarn lint
```
